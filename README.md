# DH AI Chat Widget

A beautiful floating AI chat widget for WordPress, powered by 
Claude AI (Anthropic). Engage your website visitors instantly 
with smart, branded conversations — without a single line of 
code.

## Why This Exists

Most chat widgets are either too basic or too expensive. This 
plugin brings enterprise-level AI conversation to any WordPress 
site, fully configurable and self-hosted, using your own 
Anthropic API key.

## Features

- Floating pill-style trigger button — clean, modern, unobtrusive
- Dark-themed chat window with smooth animations
- Fully powered by Claude AI — intelligent, context-aware responses
- Configurable agent name, avatar initials and brand color
- Custom system prompt — train the AI to represent your business
- Quick reply chips on first open
- Book a Free Call CTA strip built-in
- Conversation history maintained within session
- AJAX-powered — no page reload
- Mobile responsive
- Simple settings page under Settings → DH AI Chat

## Installation

1. Download and unzip the plugin
2. Upload the folder to /wp-content/plugins/
3. Activate via WordPress Admin → Plugins
4. Go to Settings → DH AI Chat
5. Enter your Anthropic Claude API key
6. Set your agent name, avatar, system prompt and brand color
7. Save — widget appears instantly on your site

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- Anthropic Claude API key — get one at console.anthropic.com

## Configuration

**Agent Name** — The name shown in the chat header. 
Example: "Deepak" or "Sarah from Support"

**System Prompt** — This is where you tell Claude who it is 
and what it knows. Example:

> You are Deepak, a senior WordPress developer. Answer questions 
> about WordPress, web design, and digital strategy. Be helpful, 
> direct and professional. For project enquiries, encourage 
> visitors to book a free call.

**Brand Color** — Matches the widget accent to your site colors

**API Key** — Your Anthropic API key, stored securely in 
WordPress options
