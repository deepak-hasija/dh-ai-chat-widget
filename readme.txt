A beautiful floating AI chat widget for WordPress, powered by Claude AI.

== Description ==

A premium floating chat widget inspired by the best SaaS products. Features:

* Beautiful pill-style trigger button (like the Arctic Grey / Mila widget)
* Dark-themed chat window with smooth animations
* Fully powered by Claude AI (Anthropic API)
* Configurable agent name, avatar, brand color, and system prompt
* Quick reply chips on first open
* Book a Free Call CTA strip built-in
* Conversation history maintained within the session
* AJAX-powered — no page reload
* Mobile responsive
* Settings page under Settings → DH AI Chat

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin in WordPress Admin → Plugins
3. Go to Settings → DH AI Chat
4. Enter your Anthropic Claude API key (get one at console.anthropic.com)
5. Customize your agent name, avatar, system prompt, and brand color
6. Save — the widget appears instantly on your site!

== Frequently Asked Questions ==

= Where do I get a Claude API key? =
Visit https://console.anthropic.com and create an account. API usage is pay-as-you-go and very affordable for chat volumes.

= Can I change the brand color? =
Yes. Go to Settings → DH AI Chat and use the color picker.

= Can I use my own photo as the avatar? =
Yes. Upload your photo to WordPress Media Library, copy the URL, and paste it in the Agent Avatar URL field.

= How do I customize what the AI knows? =
Edit the System Prompt in the settings page. This is where you tell the AI about your services, personality, and goals.
